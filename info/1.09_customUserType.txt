Представим, что у нас в БД есть тип, которого нет среди прочих в JavaSql-типах, но есть в postgres. Например JSONB.
    Следовательно нам нужно реализовать свой тип данных и преобразовать в него из массива байт.

    Добавить:
     implementation 'com.fasterxml.jackson.core:jackson-databind:2.14.0-rc2'

     /**
          * Аннотация ниже позволяет создать свой собственный кастомный тип. В данном случае String будет приводиться к формату JSON, однако
          * для корректной работы необходима зависимость для парсинга JSON (например библиотека Jackson)
          */

         @JdbcTypeCode(SqlTypes.JSON)
         private String info;
